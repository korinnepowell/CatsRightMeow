# Korinne Powell Project 4 CMST385
This is a project website showcasing my work done for a cat breeder. It is multiple pages developed to preview available kittens, 
find breed specific care information, and allows visitors to reach out through a contact form. The site 
leverages modern web tools like SASS and JavaScript and subtle design elements such as 
favicons, font packages, and animated buttons. 

## Features
- Clean layout  
- navigation bar  
- Available kittens carousel  
- favicons  
- font packages 
- animated buttons
- Contact form area (static)

## Built With
- HTML5  
- SASS  
- JavaScript  
- Favicon
